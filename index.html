HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • A tag refers to what appears inside the angle brackets
  • The p element represents a paragraph or block of text
  • The ul, ol, li elements represrnt unordered lists, ordered lists and list items

CSS

  • A margin indicates how much pace we want around the outside of the element
  • A padding indicates how much pace we want around the content outside the element
  • A link element creates the connection to the external file
  • Use special wild card symbol when apply a rule to all elements visible on the page

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to main branch
  • git pull origin main: pull down the latest main branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Console refers to a test environment that developers use to check the code.
  • The .log() method means a set of instuctions that can be executed by a computer.
  • Control flow is the order in which computer executes code in a script.
  • Computer encounters if statement in JS files and checks if the condition provided is truthy.
  • An array is a single variable that is used to hold a group of data.
